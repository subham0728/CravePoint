# CravePoint

## Current State
Full-stack ICP app with React/Tailwind frontend and Motoko backend. Features: student menu browsing with food images, cart, order placement (with name/roll number), order tracker with glowing active step, My Orders history (local storage), staff login (password: cafeteria2024), staff dashboard for managing order statuses, staff password change. All 8 menu items have food images.
