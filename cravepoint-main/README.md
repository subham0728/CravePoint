

🚀 CravePoint
Campus Cafeteria Ordering System
🧠 About the Project

CravePoint is a full-stack cafeteria ordering system designed to simplify food ordering inside a university campus.

Instead of long queues and confusion, students can:

Browse menu
Place orders
Track order status in real-time

This project is inspired by real-world systems like Domino’s and Swiggy, but adapted specifically for campus environments.

🎯 Problem It Solves

In most campuses:

Students waste time in queues
No order tracking system
Staff faces crowd mismanagement

👉 CravePoint solves this by:

Digitizing the ordering process
Providing real-time order tracking
Improving overall cafeteria efficiency
✨ Features
👨‍🎓 Student Side
Browse food items with images and prices
Add/remove items from cart
Place orders easily

Real-time order tracking

Ordered → Preparing → Ready for Pickup
Clean and modern UI (dark theme)
👨‍🍳 Staff Side
Staff login system
Dashboard to view all orders
Update order status
Change password functionality
